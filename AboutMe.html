Coding in College

Since I had taken AP Computer Science in high school, I had a general understanding of the basics of coding, which helped me a ton in the beginning stages of my coding classes at Temple University. For all classes up to data structures, it was a nice refresher on the code that I may have forgotten. If you would like to see some of those projects, you may view them here. Also, with these classes, you could have a deeper understanding of how certain things work when code is executed. For example, the different types of lists. I had learned about lists in my C class, but not much more than that. Then we learned about arraylist and linkedlists and how they are better for different functionalities. We also had classes on other types of coding, like using SQL to access database information. I loved SQL as it is a very easy language to learn and understand. The first class that was a struggle and really forced learning was Component-Based Software Design.

In this class, we would learn the language C#, which is the language I am most familiar with as I have coded in this language for almost two years now. We started with basics, like how a loop is formed, if statements, and arrays. All of these were similar to Java, the language we had used up until this class, so the assignments were not that difficult for a little while. After that, however, we started to do more complex coding assignments. In C#, there is the backend part of it, where it is C# code itself, but there is the frontend aspect of it too, where you can create GUI for the applications you are creating. Understanding how to get data from the frontend to the backend was sometimes easier said than done. The most complex project we had was a web application that was a ordering system. This included using gridviews, sessions, database connections, and C# classes, as we this class also focused on OOP (Object Oreiented Programming). We would create custom classes for many things, such as a Car, Students, and other things that could be needed for the certain project. Although we were given a fair amount of time to complete these types of projects, they were a ton of work and took a good amount of time not only code everything, but research what needed to be done with an error that would come up.
